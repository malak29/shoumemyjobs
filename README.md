# shoumemyjobs

Small static page that builds **LinkedIn job search URLs** for the **United States** and selected **states** (custom keywords, time range, company filter, optional Easy Apply). Each button opens the generated URL in a **new tab**.

## GitHub Pages

Live site (after you enable Pages once): **https://malak29.github.io/shoumemyjobs/**

1. In the repo on GitHub: **Settings** → **Pages**.
2. Under **Build and deployment** → **Source**, choose **Deploy from a branch**.
3. **Branch**: `master`, folder **/ (root)** → **Save**.

The app is served from `index.html` at the site root. You can also open `index.html` locally in a browser.
